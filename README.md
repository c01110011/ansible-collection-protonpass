# c01110011.protonpass

Ansible lookup plugin for retrieving secrets from [Proton Pass](https://proton.me/pass)
vaults via the official `pass-cli`.

> **Status:** Work in progress. Not yet published to Ansible Galaxy.

## Requirements

- [`pass-cli`](https://proton.me/pass/download) installed and authenticated
- `ansible-core >= 2.14`
